# Modification Memo - 2026-01-08

## Overview: Strategic Planning & Architecture Refactoring

This update introduces the official 2026 strategic roadmap, TinyML research materials, and implements a major refactoring of the core module architecture.

### 1. Strategic Documentation (Planning)

- **2026 Strategic Roadmap**: Created `docs/2026年度战略规划与执行方案.md`.

### 2. Core Architecture Refactoring (High Cohesion)

- **SMA (Standard Module Architecture)**:
  - Refactored `src/modules/core/tinyml.ts` from a single file into a directory-based MVC structure (`tinyml/index.ts`, `tinyml/blocks.ts`, `tinyml/generator.ts`).
  - Separated UI definitions (Blockly) from Code Generation (C++) to enforce "Separation of Concerns".
  - Updated `docs/核心开发指南.md` with Section 12 to standardize this architecture for future modules.
  - Confirmed backward compatibility with `src/modules/index.ts` imports.
