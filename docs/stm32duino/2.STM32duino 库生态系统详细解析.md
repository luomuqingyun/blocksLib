## 二、内置库详细解析与应用

### 2.1 EEPROM 库

#### 实现原理
STM32 无真实 EEPROM,通过 **Flash 最后两个扇区/页模拟**实现:
- **Page 0** 和 **Page 1** 交替使用
- 每次写入追加新记录(地址+数据)
- 当一页满时,将有效数据复制到另一页并擦除旧页
- 基于 ST AN2594 应用笔记实现

#### 核心代码位置
```
libraries/EEPROM/src/
├── EEPROM.h              # Arduino 兼容 API
└── flash_stm32.c         # Flash 擦写操作
    ├── eeprom_init()
    ├── eeprom_read_byte()
    ├── eeprom_write_byte()
    └── eeprom_buffer_fill()  # 缓冲模式
```

#### 完整应用示例

**示例 1: 基础读写**
```cpp
#include <EEPROM.h>

void setup() {
  Serial.begin(115200);
  
  // 写入单字节
  EEPROM.write(0, 123);
  
  // 读取单字节
  byte value = EEPROM.read(0);
  Serial.print("Value: ");
  Serial.println(value);
  
  // update() 只在值改变时写入(减少 Flash 损耗)
  EEPROM.update(1, 42);
}

void loop() {}
```

**示例 2: 存储结构体**
```cpp
#include <EEPROM.h>

struct Settings {
  uint32_t magic;      // 魔数,用于验证数据有效性
  float brightness;
  uint8_t mode;
  char name[20];
};

Settings config;

void setup() {
  Serial.begin(115200);
  
  // 从 EEPROM 读取设置
  EEPROM.get(0, config);
  
  // 检查数据是否有效
  if (config.magic != 0xDEADBEEF) {
    // 首次使用,初始化默认值
    config.magic = 0xDEADBEEF;
    config.brightness = 0.5;
    config.mode = 1;
    strcpy(config.name, "MyDevice");
    
    // 保存到 EEPROM
    EEPROM.put(0, config);
    Serial.println("Default settings saved");
  } else {
    Serial.println("Settings loaded:");
    Serial.print("Brightness: ");
    Serial.println(config.brightness);
    Serial.print("Mode: ");
    Serial.println(config.mode);
    Serial.print("Name: ");
    Serial.println(config.name);
  }
}

void loop() {
  // 修改设置并保存
  if (Serial.available()) {
    config.mode++;
    EEPROM.put(0, config);
    Serial.println("Settings updated");
  }
}
```

**示例 3: 高效缓冲模式**
```cpp
#include <EEPROM.h>

void setup() {
  Serial.begin(115200);
  
  // 一次性写入大量数据,减少 Flash 擦写
  eeprom_buffer_fill();  // 从 Flash 加载到 RAM
  
  for (int i = 0; i < 100; i++) {
    eeprom_buffered_write_byte(i, i * 2);  // 在 RAM 中修改
  }
  
  eeprom_buffer_flush();  // 一次性写回 Flash
  
  Serial.println("100 bytes written efficiently");
}

void loop() {}
```

**注意事项**:
- Flash 擦写次数有限(约 10,000-100,000 次)
- 擦除操作可能耗时 20-200ms,会阻塞代码
- 使用 `update()` 而非 `write()` 可延长寿命
- 频繁数据更新应考虑外部 EEPROM 芯片

### 2.2 Servo 库

#### 实现原理
- 使用 **软件生成 PWM 信号**,可在任意 GPIO 使用
- 基于单个硬件定时器(默认 `TIMER_SERVO`)产生时基
- 在中断中顺序生成多个舵机的控制信号
- 支持最多 12 个舵机

#### 核心代码位置
```
libraries/Servo/src/
├── Servo.h
└── stm32/
    └── ServoTimers.h      # 定义 TIMER_SERVO
```

#### 工作原理图解
```
定时器中断时序:
     |<--- 20ms 周期 --->|
     
Servo1: |----1-2ms----|______________|
              ↑
           信号长度决定角度
              
Servo2: _____________|----1-2ms----|_
              
Servo3: ____________________________|--
```

#### 完整应用示例

**示例 1: 单舵机控制**
```cpp
#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(PA0);  // 任意 GPIO 引脚
  myServo.write(90);    // 设置到中间位置
}

void loop() {
  // 扫描 0° 到 180°
  for (int angle = 0; angle <= 180; angle++) {
    myServo.write(angle);
    delay(15);
  }
  
  // 扫描回 0°
  for (int angle = 180; angle >= 0; angle--) {
    myServo.write(angle);
    delay(15);
  }
}
```

**示例 2: 多舵机机械臂**
```cpp
#include <Servo.h>

Servo base, shoulder, elbow, wrist;

void setup() {
  base.attach(PA0);
  shoulder.attach(PA1);
  elbow.attach(PA2);
  wrist.attach(PA3);
  
  // 初始位置
  resetPosition();
}

void resetPosition() {
  base.write(90);
  shoulder.write(90);
  elbow.write(90);
  wrist.write(90);
}

void pickObject() {
  // 复杂动作序列
  shoulder.write(45);    // 俯身
  delay(500);
  elbow.write(120);      // 伸臂
  delay(500);
  wrist.write(60);       // 夹取
  delay(500);
  shoulder.write(90);    // 抬起
  delay(500);
}

void loop() {
  pickObject();
  delay(2000);
  resetPosition();
  delay(2000);
}
```

**示例 3: 自定义脉冲宽度**
```cpp
#include <Servo.h>

Servo myServo;

void setup() {
  // 自定义脉冲范围: 500-2500μs (而非默认 1000-2000μs)
  myServo.attach(PA0, 500, 2500);
}

void loop() {
  // 使用微秒直接控制
  myServo.writeMicroseconds(1500);  // 中间位置
  delay(1000);
  myServo.writeMicroseconds(1000);  // 一端
  delay(1000);
  myServo.writeMicroseconds(2000);  // 另一端
  delay(1000);
}
```

**性能优化**: 如需高精度 PWM,可直接使用 HardwareTimer:
```cpp
HardwareTimer *servoTimer = new HardwareTimer(TIM2);

void setup() {
  // 使用硬件 PWM (仅限定时器输出引脚)
  servoTimer->setPWM(1, PA0, 50, 7.5);  // 50Hz, 7.5% (1.5ms)
}
```

### 2.3 SoftwareSerial 库

#### 实现原理
- 使用**硬件定时器中断**实现软件 UART
- 中断频率 = 波特率 × OVERSAMPLE(默认 3)
- 例如 9600 bps → 28,800 Hz 中断频率
- 高波特率会显著增加 CPU 负载

#### 代码位置
```
libraries/SoftwareSerial/src/
├── SoftwareSerial.h
└── SoftwareSerial.cpp    # 定时器中断处理
```

#### 应用示例

**示例 1: 基础软件串口**
```cpp
#include <SoftwareSerial.h>

SoftwareSerial mySerial(PA10, PA9); // RX, TX

void setup() {
  Serial.begin(115200);      // 硬件串口
  mySerial.begin(9600);      // 软件串口
}

void loop() {
  // 从软件串口读取并转发到硬件串口
  if (mySerial.available()) {
    char c = mySerial.read();
    Serial.write(c);
  }
  
  // 从硬件串口读取并转发到软件串口
  if (Serial.available()) {
    char c = Serial.read();
    mySerial.write(c);
  }
}
```

**示例 2: 多设备通信**
```cpp
#include <SoftwareSerial.h>

SoftwareSerial gps(PA10, PA9);      // GPS 模块
SoftwareSerial bluetooth(PB6, PB7); // 蓝牙模块

void setup() {
  Serial.begin(115200);
  gps.begin(9600);
  bluetooth.begin(9600);
}

void loop() {
  // 读取 GPS 数据
  if (gps.available()) {
    String gpsData = gps.readStringUntil('\n');
    Serial.print("GPS: ");
    Serial.println(gpsData);
    
    // 通过蓝牙转发
    bluetooth.println(gpsData);
  }
}
```

**性能建议**:
- 优先使用硬件 UART (Serial1, Serial2...)
- SoftwareSerial 波特率建议 ≤ 19200
- 高波特率考虑使用硬件 UART 或 USART

### 2.4 IWatchdog 库(独立看门狗)

#### 实现原理
- 使用 STM32 独立看门狗(IWDG)外设
- 超时未喂狗则自动复位系统
- 用于检测程序死锁/崩溃

#### 代码位置
```
libraries/IWatchdog/src/IWatchdog.h
```

#### 应用示例

**示例 1: 基础看门狗**
```cpp
#include <IWatchdog.h>

void setup() {
  Serial.begin(115200);
  
  // 初始化看门狗,超时 4 秒
  IWatchdog.begin(4000000);  // 微秒
  
  Serial.println("Watchdog started");
}

void loop() {
  // 正常运行时定期喂狗
  IWatchdog.reload();
  
  Serial.println("Working...");
  delay(1000);
}
```

**示例 2: 死锁检测**
```cpp
#include <IWatchdog.h>

void setup() {
  Serial.begin(115200);
  
  // 检查是否是看门狗复位
  if (IWatchdog.isReset()) {
    Serial.println("System was reset by watchdog!");
    IWatchdog.clearReset();
  }
  
  IWatchdog.begin(2000000);  // 2 秒超时
}

void loop() {
  // 模拟可能的死锁
  if (Serial.available()) {
    char c = Serial.read();
    if (c == 'd') {
      Serial.println("Entering deadlock...");
      while(1);  // 故意死锁,2 秒后系统重启
    }
  }
  
  IWatchdog.reload();
  delay(500);
}
```

## 三、外部专用库详解

### 3.1 STM32Ethernet 库

#### 功能说明
为带有以太网外设的 STM32(如 F4/F7/H7 系列)提供网络支持,完全兼容 Arduino Ethernet API。

#### 依赖库
- **LwIP**: 轻量级 TCP/IP 协议栈

#### 应用示例

**示例 1: Web 服务器**
```cpp
#include <LwIP.h>
#include <STM32Ethernet.h>

EthernetServer server(80);

void setup() {
  Serial.begin(115200);
  
  // 配置静态 IP
  IPAddress ip(192, 168, 1, 100);
  Ethernet.begin(ip);
  
  server.begin();
  Serial.print("Server at ");
  Serial.println(Ethernet.localIP());
}

void loop() {
  EthernetClient client = server.available();
  
  if (client) {
    String request = client.readStringUntil('\r');
    Serial.println(request);
    
    // HTTP 响应
    client.println("HTTP/1.1 200 OK");
    client.println("Content-Type: text/html");
    client.println();
    client.println("<h1>STM32 Web Server</h1>");
    client.println("<p>Temperature: 25°C</p>");
    
    client.stop();
  }
}
```

**示例 2: HTTP 客户端**
```cpp
#include <LwIP.h>
#include <STM32Ethernet.h>

EthernetClient client;

void setup() {
  Serial.begin(115200);
  Ethernet.begin();
  
  delay(1000);
  
  // 连接到服务器
  if (client.connect("api.weather.com", 80)) {
    client.println("GET /data/weather HTTP/1.1");
    client.println("Host: api.weather.com");
    client.println();
  }
}

void loop() {
  if (client.available()) {
    char c = client.read();
    Serial.write(c);
  }
  
  if (!client.connected()) {
    client.stop();
    while(1);
  }
}
```

### 3.2 STM32SD 库

#### 功能说明
支持 SDIO/SDMMC 接口的 SD 卡操作,兼容 Arduino SD API。

#### 依赖库
- **FatFs**: FAT 文件系统

#### 应用示例

**示例 1: 数据记录**
```cpp
#include <STM32SD.h>

File dataFile;

void setup() {
  Serial.begin(115200);
  
  if (!SD.begin()) {
    Serial.println("SD Card failed!");
    return;
  }
  
  // 创建/打开文件
  dataFile = SD.open("datalog.txt", FILE_WRITE);
  
  if (dataFile) {
    dataFile.println("Timestamp,Temperature,Humidity");
    dataFile.close();
  }
}

void loop() {
  // 追加数据
  dataFile = SD.open("datalog.txt", FILE_WRITE);
  
  if (dataFile) {
    dataFile.print(millis());
    dataFile.print(",");
    dataFile.print(analogRead(A0) * 0.1);
    dataFile.print(",");
    dataFile.println(analogRead(A1) * 0.05);
    dataFile.close();
  }
  
  delay(1000);
}
```

**示例 2: 配置文件读取**
```cpp
#include <STM32SD.h>

struct Config {
  int mode;
  float threshold;
  char ssid[32];
};

Config loadConfig() {
  Config cfg = {0, 0.0, ""};
  
  File configFile = SD.open("config.txt", FILE_READ);
  if (configFile) {
    // 简单解析
    String line;
    while (configFile.available()) {
      line = configFile.readStringUntil('\n');
      
      if (line.startsWith("mode=")) {
        cfg.mode = line.substring(5).toInt();
      } else if (line.startsWith("threshold=")) {
        cfg.threshold = line.substring(10).toFloat();
      } else if (line.startsWith("ssid=")) {
        line.substring(5).toCharArray(cfg.ssid, 32);
      }
    }
    configFile.close();
  }
  
  return cfg;
}

void setup() {
  Serial.begin(115200);
  SD.begin();
  
  Config config = loadConfig();
  Serial.print("Mode: ");
  Serial.println(config.mode);
}

void loop() {}
```

### 3.3 STM32RTC 库

#### 功能说明
支持 STM32 内部实时时钟(RTC),提供时间/日期/闹钟功能。

#### 应用示例

**示例 1: 时间设置与读取**
```cpp
#include <STM32RTC.h>

STM32RTC& rtc = STM32RTC::getInstance();

void setup() {
  Serial.begin(115200);
  
  // 选择时钟源(LSE 外部晶振最精确)
  rtc.setClockSource(STM32RTC::LSE_CLOCK);
  rtc.begin();
  
  // 设置时间: 2024-01-15 10:30:45
  rtc.setDate(15, 1, 24);  // day, month, year(2000+)
  rtc.setTime(10, 30, 45); // hour, min, sec
}

void loop() {
  // 读取并显示时间
  Serial.printf("%02d:%02d:%02d ", 
                rtc.getHours(), 
                rtc.getMinutes(), 
                rtc.getSeconds());
  Serial.printf("%04d-%02d-%02d\n",
                rtc.getYear() + 2000,
                rtc.getMonth(),
                rtc.getDay());
  
  delay(1000);
}
```

**示例 2: 闹钟唤醒**
```cpp
#include <STM32RTC.h>

STM32RTC& rtc = STM32RTC::getInstance();

void alarmCallback(void *data) {
  Serial.println("Alarm triggered!");
  digitalWrite(LED_BUILTIN, !digitalRead(LED_BUILTIN));
}

void setup() {
  Serial.begin(115200);
  pinMode(LED_BUILTIN, OUTPUT);
  
  rtc.setClockSource(STM32RTC::LSE_CLOCK);
  rtc.begin();
  
  // 当前时间
  rtc.setTime(10, 0, 0);
  
  // 设置闹钟: 5 秒后触发
  rtc.attachInterrupt(alarmCallback);
  rtc.setAlarmTime(10, 0, 5);
  rtc.enableAlarm(rtc.MATCH_SS);  // 只匹配秒
}

void loop() {
  // 主循环可以休眠,闹钟会唤醒
}
```

**示例 3: Unix 时间戳**
```cpp
#include <STM32RTC.h>

STM32RTC& rtc = STM32RTC::getInstance();

void setup() {
  Serial.begin(115200);
  rtc.begin();
  
  // 设置 Unix 时间戳(从 2000-01-01 开始)
  uint32_t epoch = 1705392000;  // 2024-01-16 00:00:00
  rtc.setEpoch(epoch);
}

void loop() {
  uint32_t now = rtc.getEpoch();
  Serial.print("Epoch: ");
  Serial.println(now);
  
  delay(1000);
}
```

### 3.4 STM32LowPower 库

#### 功能说明
支持 STM32 的低功耗模式(Sleep, Stop, Standby)。

#### 应用示例

**示例: 定时唤醒**
```cpp
#include <STM32LowPower.h>

void setup() {
  Serial.begin(115200);
  LowPower.begin();
}

void loop() {
  Serial.println("Going to sleep for 10 seconds...");
  Serial.flush();
  
  // 进入深度睡眠模式
  LowPower.deepSleep(10000);  // 10 秒
  
  Serial.println("Woke up!");
  
  // 执行快速任务
  digitalWrite(LED_BUILTIN, HIGH);
  delay(100);
  digitalWrite(LED_BUILTIN, LOW);
}
```

## 四、库使用最佳实践

### 4.1 内存管理
```cpp
// 避免在 EEPROM 中存储大对象
struct LargeData {
  uint8_t buffer[2048];  // 太大!
};

// 更好的方法:
struct SmallData {
  uint32_t crc;      // 校验和
  uint16_t size;     // 实际数据大小
  uint8_t key[16];   // 关键配置
};
```

### 4.2 错误处理
```cpp
#include <STM32SD.h>

void safeWrite(const char* filename, const char* data) {
  File file = SD.open(filename, FILE_WRITE);
  
  if (!file) {
    Serial.println("Failed to open file");
    return;
  }
  
  size_t written = file.print(data);
  file.close();
  
  if (written == 0) {
    Serial.println("Write failed");
  }
}
```

## 五、总结

STM32duino 库生态系统通过以下方式实现了功能丰富且易用的开发环境:

1. **内置库**位于 `Arduino_Core_STM32/libraries/`,提供基础功能
2. **外部库**作为独立仓库,提供专业功能(网络、存储、传感器等)
3. **底层实现**基于 STM32 HAL/LL,充分利用硬件
4. **Arduino 兼容**的 API 设计降低学习曲线

通过这些库,开发者可以快速实现从简单 LED 控制到复杂物联网应用的各种项目!
