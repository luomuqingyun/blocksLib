# 关于 EmbedBlocks Studio

**EmbedBlocks Studio** 是下一代专为嵌入式系统设计的可视化编程环境。我们的使命是弥合复杂硬件控制与直观可视化逻辑之间的鸿沟，让学生、创客和专业工程师都能轻松进行嵌入式开发。

---

## 🚀 核心理念

- **简约之美**: 现代、干净的界面，提供强大工具的同时不干扰您的工作流。
- **工业级生态**: 原生支持 **1,430** 款 STM32 芯片型号，覆盖从 C0 到 H7 的全系列产品线。采用严格的官方支持库过滤，确保 100% 变体准确性。
- **插件优先**: 开放式架构，允许社区轻松扩展新的积木和板卡定义。

---

## ✨ 核心特性

- **可视化编程**：基于 Google Blockly，提供直观的积木式逻辑构建界面。
- **双模式编辑器**：可在可视化积木与专业 C++ 编辑器（基于 Monaco Editor）之间无缝切换。
- **高度集成工具链**：内置 PlatformIO 驱动，支持串口监视、开发板配置及固件上传。
- **智能帮助系统**：内置 Encyclopedia 级用户手册与插件开发指南，支持 Markdown 全文检索与代码预览。
- **原生多语言支持**：全量 UI 国际化（中/英），支持插件运行时动态翻译加载。
- **AI 智能助手 (v1.4.1 更新)**：专为硬件开发的 24/7 AI 专家，具备上下文代码感知能力，通过深度布局优化实现极致的排版渲染。
- **实时芯片视觉预览**：集成官方引脚映射数据，能够为 **1,430** 款 STM32 芯片渲染精确到物理位置的 SVG 交互式图表。
- **数据驱动的硬件引擎 (Board Data V2)**: 仅需 JSON 配置即可定义新板卡，支持动态引脚复用提示与外设约束。
- **高级插件系统**：支持内置 C++ 库文件夹自动编译，兼容多平台硬件驱动。

## 🏗️ 架构亮点

项目采用面向服务的解耦架构，并在 **2026 年初完成了架构 2.0 重构**：

- **Hook-based Architecture**: 业务逻辑全面解耦，基于 Custom Hooks 提供高内聚的 State 管理。
- **Auto-Discovery**: 工业级的数据自动化流水线，通过集成 ST 官方、Embassy-rs 和 STM32duino 三大数据源，实现芯片定义与物理布局的自动补全。
- **CodeBuilder Engine**: 自研的 C++ 代码组装引擎，支持智能去重与依赖分析。
