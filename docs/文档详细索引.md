# EmbedBlocks 文档中心 (Documentation Center)

根据您的角色，请选择相应的文档：

### 🛠️ 我是核心开发者 (Internal Developer)
> 我有项目源码，我要修改核心功能，或者把扩展永久集成到软件里。
*   **[内部开发插件指南 (Extension Dev Guide)](./内部开发人员插件开发指南.md)**: 学习项目架构、SMA 模式以及数据自动发现机制。
*   **[协作公约 (Collaboration Agreement)](../PROJECT_RULES.md)**: 开发者共同遵守的语言与习惯约定。
*   **[重构指南 (Refactoring Guide)](./旧版模式重构指南.md)**: 了解老旧代码的迁移规范。
*   **[修改记录 (Modification Memo)](./历史修改备忘录.md)**: 查看项目演进历史。
*   **[自定义板卡 SVG 视图规范指南](./自定义板卡_SVG_视图规范指南.md)**: (核心/内部) 学习如何制作具备动态交互能力的 1:1 实物级开发板视图。
*   **[开发历程 (Dev History)](./开发历程回顾.md)**: 查看完整技术演进。

### 🧩 我是插件创作者 (Extension Creator)
> 我没有源码（或不想编译），我只想做一个 `.zip` 包发给别人用。
*   **[用户插件开发指南 (User Extension)](./普通用户插件系统开发手册.md)**: 适合第三方开发者的标准手册，学习如何编写 `blocks.json` 和 `generator.js`。
*   **[新手教程-添加新板卡](./新手教程-添加新板卡.md)**: 了解如何通过 JSON 快速添加新硬件支持。

*   **[STM32WBA系列支持说明](./stm32duino/STM32WBA系列支持说明.md)**: 针对 WBA 系列 (Cortex-M33) 的专项构建修复方案及使用指南。
*   **[STM32WL系列支持说明](./stm32duino/STM32WL系列支持说明.md)**: 针对 WL/WLE 无线 MCU 的变体冲突过滤与双核启动说明。
*   **[未支持芯片说明](./stm32duino/未支持芯片说明.md)**: 了解目前尚未适配的型号及其原因（WBA50 已支持）。
*   **[不受支持的 8KB 闪存芯片黑名单](./unsupported_8kb_chips.md)**: 由于底层框架编译体积限制而被系统硬拦截无法创建工程的超小容量板卡清单。
